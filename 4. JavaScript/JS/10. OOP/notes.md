# javascript and classes

### Does Javascript really  has classes?

Yes, JavaScript has classes, introduced in ECMAScript 6 (ES6) in 2015. They provide a cleaner syntax for creating objects and handling inheritance, though they are syntactic sugar over JavaScript's prototype-based inheritance.
